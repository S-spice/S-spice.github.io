Transformer Fault Diagnosis with CA-CNN

A deep learning project for transformer fault diagnosis using convolutional attention mechanisms and dissolved-gas analysis features.

Problem

Transformer fault diagnosis requires extracting informative patterns from dissolved-gas and condition-monitoring data under noisy and imbalanced conditions.

Approach

Developed a CA-CNN model to learn diagnostic features and improve classification performance compared with conventional machine-learning baselines.

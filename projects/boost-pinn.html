Physics-informed Modeling for DC-DC Boost Converter Dynamics

A self-directed project on converter dynamic prediction under load and input disturbances using Simulink data generation and physics-informed sequence modeling.

Motivation

Pure data-driven models can learn converter dynamics from simulated or experimental waveforms, but their generalization may degrade under parameter shifts or unseen disturbances. This project explores whether lightweight physics-informed constraints can improve transient prediction for closed-loop DC-DC converters.

System

The initial system is a closed-loop Boost converter simulated in MATLAB/Simulink. The model includes input voltage variations, load steps, and parameter perturbations in inductance and capacitance.

Method

  • Build and validate a closed-loop Boost converter simulation.
  • Generate time-series datasets under load, input, and parameter disturbances.
  • Train baseline LSTM/TCN models for output-voltage and inductor-current prediction.
  • Add physics-informed loss terms based on averaged converter equations.
  • Compare in-distribution and out-of-distribution prediction performance.

Status

Planning stage. The first milestone is a stable Simulink model with batch data export.
